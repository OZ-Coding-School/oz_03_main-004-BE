# Potata Project
GitHub와 백준 활동으로 성장하는 감자 캐릭터를 키우며 생산성을 관리하는 게임형 웹 앱입니다.

## 기능
- 사용자 감자 목록 조회, 수정, 업데이트
- 사용자 스택 조회, 수정, 업데이트
- 캘린더 조회
- Todo 조회, 추가, 수정, 삭제
- 회원가입$
- 소셜 로그인, 깃허브, 백준 연동
- 테스트 코드 (추가 예정)
- 감자 코인 (추가 예정)
- 감자 상점 (추가 예정)
- 감자 스킨 (추가 예정)
